# como configurar arch linux y las cosas raras de hyprland

## 1. basicos de arch linux

actualizar sistema: 
1. sudo pacman -Syu
2. mkinitcpio -U              <- MUY IMPORTANTE

apagar:
1. poweroff

reiniciar:
1. reboot

instalar cosas:
1. sudo pacman -S <nombre del paquete>

desinstalar cosas:
1. sudo pacman -R <nombre del paquete>

## 2. hyprland bldooo

modificar la configuracion:
1. nano ~/.config/hypr/hyprland.conf

preguntale a claude como lo puedes configurar bonito y tal

### fondos de pantalla para hyprland
1. nano ~/.config/hypr/hyprpaper.conf

tmb preguntale a claude como lo puedes configurar con tus monitores y tal


## 3. ly blyatt

1. sudo nano /etc/ly/config.ini

puedes preguntarle a claud como configurarlo tambien

## 4. alacritty
